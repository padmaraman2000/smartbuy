# ✈️ SmartBuy — Corporate Travel Booking App

A complete corporate travel booking demo app with Flight, Hotel, Bus, and Train flows.

## 🗺️ Complete User Flows

### ✈️ Flight Flow
Login → Dashboard → Flight Search → Flight Listing → Flight Info → Baggage Policy → Cancellation Policy → GoDigit Insurance → Customer Details → Booking Request → Booking Response → E-Ticket

### 🏨 Hotel Flow
Dashboard → Hotel Search → Hotel Listing (with amenities, ratings) → Booking Confirmation

### 🚌 Bus Flow
Dashboard → Bus Search → Bus Listing → Seat Selection Map → Booking Confirmation

### 🚂 Train Flow
Dashboard → Train Search → Train Listing (multiple classes) → Booking Confirmation

## ✨ Features

- 🔐 Corporate login with company code
- 📋 Travel policy banner (economy class, budget limits)
- ✈️ Flight comparison with tags (cheapest, fastest, refundable)
- 🧳 Detailed baggage policy table
- ❌ Cancellation & refund policy
- 🛡️ GoDigit insurance plan selection (Basic/Standard/Premium)
- 👤 Passenger details form with GST option
- 📤 Booking request payload (JSON view)
- 📥 Booking response payload (JSON view with PNR)
- 💺 Interactive bus seat map
- 🎫 E-ticket confirmation page
